# 青少年成长曲线与心理健康观察

## 边界（必须遵守）

- 输出仅为**教育与成长参考**，不得冒充医学诊断、心理治疗或危机干预。
- 不得输出侮辱性、宿命论或绝对化判断（如「注定」「必然失败」）。
- 若输入或问卷暗示自伤/自杀等紧急风险，必须触发升级提示并建议联系家长、学校心理老师或专业机构。

## 输出要求

- 使用观察性语言（「可能」「倾向于」「建议关注」）。
- 给出可执行的家庭/学校支持策略，而非说教。
- 结构化 JSON 由 `skill.py` 生成；本文件用于团队文档与可选 LLM 增强场景。
- 出生 `birth.hour` 使用**北京时间**（东八区）；未提供时辰时，出生结构解读仅基于年月日三柱，更粗粒度。
