# 参考知识库（摘录整理）

本文件由项目内两份 Word 资料（儿童五行成长规划、盲派溯源版与「文档2」心理—曲线对照）**提炼为可检索的教育文本**，用于 Skill 生成报告时的语境一致性。**不等同于命理推演结论**，实现层使用确定的启发式规则与静态曲线矩阵。

## 1. 五行与心理特质（观察性标签）

- **水系**：思维快、内耗与思虑倾向；压力下易摇摆；理科与逻辑兴趣常见。
- **金系**：重规则、秩序与完美主义；受挫时易僵硬；语言与精密任务常见。
- **土系**：稳健、包容、节奏偏慢；压力下易闷在心里；长期积累型任务常见。
- **木系**：共情与探索欲强；压力下易急躁或退缩；人文与创意表达常见。
- **火系**：外向表现欲强；冲动与焦躁风险；演讲、运动与团队舞台常见。

## 2. 年度曲线（2026–2031）使用说明

曲线矩阵将每年映射为：

- `psych_state`：心理状态与动能的**观察性描述**（非预测命运）。
- `learning_focus` / `joy_index` / `wellbeing_hint`：1–5 的**相对比较刻度**，用于识别相对高峰/低谷年份。
- `concern` / `guidance`：身心联动提示与教养策略（非临床诊断）。

## 3. 逆境干预原则（教育场景）

- **高压期**：减少羞辱式批评；任务拆解；睡眠优先；保留运动与社交出口。
- **逆反/对抗增强**：给可控选择权；把「对错辩论」转成「共同解决问题」。
- **退缩/回避增强**：先恢复安全感与启动步骤（5 分钟启动法）；降低初期目标。
- **好胜/冒险增强**：竞技与体育出口；强调复盘与规则；预防人际冲突升级。

## 4. 中医健康提示（仅供参考）

文档中含有脏腑与体质联动的**传统文化表述**，本工具仅作为「身心同治的生活照护提示」，不能替代医疗评估；不适请就医。

## 5. 与计算层的分工

- **分型与问卷**：由 `util/youth_growth` 计算（可解释、可单测）。
- **年度曲线**：由 `curve_engine` 的静态矩阵提供（与五行 key 对齐）。
- **危机关键词**：由 `safety` 模块扫描（可迭代词表）。
